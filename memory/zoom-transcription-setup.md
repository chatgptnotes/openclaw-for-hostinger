## Required Zoom Settings (verify in zoom.us/profile/setting)

### 1. Cloud Recording (Required for transcription)
- Go to: Settings → Recording
- Enable: "Cloud recording"
- Enable: "Record active speaker with shared screen"

### 2. Audio Transcript (Critical)
- Enable: "Audio transcript"
- This auto-generates transcripts for cloud recordings

### 3. Auto-Email Settings
- Enable: "Notify host when cloud recording is available"
- Consider: Setting up automatic email of recordings

### 4. During Meeting
- Host must click "Record to Cloud" (or enable auto-record)
- Transcription happens automatically if enabled

## Post-Meeting Workflow
1. Wait 10-15 minutes after meeting ends (processing time)
2. Go to: zoom.us → Recordings → Cloud Recordings
3. Click on meeting → Download transcript (.vtt or .txt)
4. Email to cmd@hopehospital.com
5. File in NABH evidence folder
