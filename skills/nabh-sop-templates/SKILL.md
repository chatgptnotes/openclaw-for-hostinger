# NABH SOP Templates

## Overview

This skill generates standardized Standard Operating Procedure (SOP) templates for NABH hospital accreditation compliance. It provides consistent formatting, proper structure, and comprehensive coverage for all hospital processes and clinical protocols.

## Core Capabilities

### 1. SOP Template Generation
Generate standardized SOP templates based on NABH requirements:
- Clinical procedure SOPs
- Administrative process SOPs  
- Emergency response SOPs
- Quality assurance SOPs
- Safety and infection control SOPs

### 2. SOP Categories

**Clinical SOPs:**
- Patient care protocols
- Medical procedures and interventions
- Medication administration procedures
- Infection control protocols
- Emergency response procedures

**Administrative SOPs:**
- Patient admission and discharge
- Record management procedures
- Equipment maintenance protocols
- Staff management procedures
- Quality improvement processes

**Departmental SOPs:**
- Department-specific procedures
- Inter-departmental coordination
- Communication protocols
- Escalation procedures
- Handover protocols

### 3. SOP Structure

Each SOP template includes:
- **Header Information**: SOP title, ID, version, effective date
- **Purpose and Scope**: Clear objectives and applicability
- **Responsibility**: Who is responsible for execution and monitoring
- **Procedure Steps**: Detailed step-by-step instructions
- **Documentation**: Required forms, records, and monitoring
- **Quality Control**: Monitoring and evaluation mechanisms
- **References**: Related documents, guidelines, and standards

## Quick Start

1. **Generate Clinical SOP:**
   ```
   Create NABH SOP for [Clinical Procedure/Protocol]
   ```

2. **Generate Administrative SOP:**
   ```
   Create SOP for [Administrative Process] with NABH compliance
   ```

3. **Department-Specific SOP:**
   ```
   Generate [Department] SOP for [Specific Process] following NABH standards
   ```
